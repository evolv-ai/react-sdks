# Function: useVariable()

> **useVariable**\<`T`\>(`key`, `initialState`): `T`

Hook that returns the variant allocated to the current user for the given variable.

## Type Parameters

### T

`T` = `any`

## Parameters

### key

`string`

Name of variable

### initialState

`T`

Value to use before Evolv initializes or in the event of an error

## Returns

`T`
