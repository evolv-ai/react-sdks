# Function: useConfig()

> **useConfig**\<`T`\>(`key`, `initialState`): `T`

Hook that returns the value of the given configuration key.

## Type Parameters

### T

`T` = `any`

## Parameters

### key

`string`

Name of configuration value

### initialState

`T`

Value to use before Evolv initializes or in the event of an error

## Returns

`T`
